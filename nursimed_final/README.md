# 🏥 نرسیمد - پلتفرم جامع آموزش پزشکی

یک وب‌سایت مدرن و حرفه‌ای برای فروش کتاب‌های پزشکی، مقالات آموزشی، دوره‌ها و پادکست‌ها

## 🚀 ویژگی‌ها

- ✅ UI مدرن و ریسپانسیو با انیمیشن‌های زیبا
- ✅ سیستم ثبت نام و ورود کامل
- ✅ سبد خرید پویا
- ✅ پنل ادمین اختصاصی
- ✅ Toast notifications
- ✅ کارت‌های محصول و مقاله زیبا
- ✅ دیتابیس PostgreSQL با داده‌های نمونه

## 📦 نصب و راه‌اندازی

### 1. نصب پیش‌نیازها

```bash
pip install psycopg2-binary
```

### 2. تنظیم دیتابیس PostgreSQL

```sql
-- در psql یا pgAdmin اجرا کنید:
CREATE DATABASE nursimed_db;
```

### 3. تنظیمات اتصال به دیتابیس

فایل `db_config.py` را باز کنید و رمز عبور پستگرس خود را وارد کنید:

```python
DB_CONFIG = {
    'host': 'localhost',
    'port': 5432,
    'database': 'nursimed_db',
    'user': 'postgres',
    'password': 'YOUR_PASSWORD_HERE'  # <-- رمز خود را اینجا بگذارید
}
```

### 4. ایجاد جداول و داده‌های اولیه

```bash
cd /workspace/nursimed_final
python init_db.py
```

این کار:
- تمام جداول مورد نیاز را می‌سازد
- اکانت ادمین را ایجاد می‌کند
- محصولات، مقالات، دوره‌ها و پادکست‌های نمونه را اضافه می‌کند

### 5. اجرای سرور

```bash
python app.py
```

سپس در مرورگر باز کنید:
```
http://localhost:8000
```
