# راهنمای اجرای سایت نرسیمد

## پیش‌نیازها

1. نصب پایتون (نسخه 3.x)
2. نصب PostgreSQL
3. نصب کتابخانه psycopg2 برای اتصال به دیتابیس

## مراحل راه‌اندازی

### 1. ساخت دیتابیس در PostgreSQL

ابتدا باید PostgreSQL را نصب کنید و یک دیتابیس بسازید:

```sql
-- وارد PostgreSQL شوید
psql -U postgres

-- ساخت دیتابیس
CREATE DATABASE nursimed_db;

-- خروج
\q
```

### 2. تنظیمات دیتابیس

فایل `db_config.py` را باز کنید و مشخصات دیتابیس خود را وارد کنید:

```python
DB_CONFIG = {
    'host': 'localhost',
    'database': 'nursimed_db',  # نام دیتابیسی که ساختید
    'user': 'postgres',         # نام کاربری پستگرس
    'password': 'your_password', # رمز عبور شما
    'port': '5432'
}
```

**توجه:** برای تغییر مقادیر دیتابیس، فقط کافیست این فایل را ویرایش کنید.

### 3. نصب کتابخانه مورد نیاز

```bash
pip install psycopg2-binary
```

### 4. اجرای سایت

برای اجرای سایت، کافیست دستور زیر را در ترمینال اجرا کنید:

```bash
python app.py
```

سپس مرورگر را باز کنید و به آدرس زیر بروید:
```
http://localhost:8000
```

## ساختار فایل‌ها

```
/workspace/
├── app.py              # فایل اصلی برنامه (سرور پایتون)
├── db_config.py        # تنظیمات دیتابیس (اینجا را ویرایش کنید)
├── templates/          # فایل‌های HTML
│   ├── index.html      # صفحه اصلی
│   ├── articles.html   # صفحه مقالات
│   ├── article_detail.html  # صفحه تک مقاله
│   ├── store.html      # صفحه فروشگاه
│   ├── product_detail.html  # صفحه تک محصول
│   ├── about.html      # درباره ما
│   ├── contact.html    # تماس با ما
│   ├── team.html       # اعضای تیم
│   ├── login.html      # ورود
│   ├── register.html   # ثبت نام
│   ├── 400.html        # صفحه خطای 400
│   ├── 404.html        # صفحه خطای 404
│   └── 500.html        # صفحه خطای 500
└── static/             # فایل‌های استاتیک
    ├── css/
    │   └── style.css   # استایل‌های سایت
    ├── js/
    │   └── main.js     # جاوااسکریپت سایت
    └── images/         # تصاویر سایت
```

## صفحات سایت

- **صفحه اصلی**: نمایش معرفی خدمات و آخرین مقالات
- **مقالات**: لیست تمام مقالات پزشکی
- **تک مقاله**: نمایش جزئیات هر مقاله
- **فروشگاه**: لیست کتاب‌ها، دوره‌ها و پادکست‌ها
- **تک محصول**: نمایش جزئیات هر محصول
- **درباره ما**: توضیحات درباره نرسیمد
- **تماس با ما**: فرم تماس و اطلاعات ارتباطی
- **اعضای تیم**: معرفی تیم نرسیمد
- **ورود**: فرم ورود کاربران
- **ثبت نام**: فرم ثبت نام کاربران جدید
- **صفحات خطا**: 400، 404، 500
