# سایت نرسیمد - پلتفرم آموزشی پزشکی

یک وب‌سایت آموزشی پزشکی با قابلیت‌های فروش کتاب، مقاله، پادکست و دوره‌های آموزشی.

## ویژگی‌ها

- ✅ صفحه اصلی با طراحی مدرن
- ✅ فروشگاه کتاب‌های پزشکی با سبد خرید
- ✅ سیستم مقالات پزشکی
- ✅ صفحات درباره ما، تماس با ما، اعضای تیم
- ✅ سیستم ثبت نام و ورود ساده (فقط ایمیل، نام کاربری و رمز)
- ✅ سبد خرید پویا با امکان افزودن، حذف و تغییر تعداد
- ✅ پرداخت سفارشات
- ✅ صفحات خطا 400 و 500
- ✅ طراحی ریسپانسیو و زیبا
- ✅ اتصال به دیتابیس PostgreSQL

## ساختار پروژه

```
nursimed_v2/
├── app.py              # سرور اصلی پایتون
├── db_config.py        # تنظیمات دیتابیس
├── templates/          # قالب‌های HTML (در خود app.py هستند)
└── static/
    ├── css/
    │   └── style.css   # استایل‌های سایت
    ├── js/
    │   └── main.js     # کدهای جاوااسکریپت
    └── images/         # تصاویر (اختیاری)
```

## نصب و راه‌اندازی

### 1. نصب پیش‌نیازها

```bash
# نصب کتابخانه psycopg2 برای اتصال به PostgreSQL
pip install psycopg2-binary
```

### 2. تنظیم دیتابیس PostgreSQL

1. PostgreSQL را نصب کنید (اگر نصب نیست)
2. یک دیتابیس جدید بسازید:

```sql
CREATE DATABASE nursimed_db;
```

3. فایل `db_config.py` را ویرایش کنید و رمز عبور خود را وارد کنید:

```python
DB_CONFIG = {
    'host': 'localhost',
    'port': 5432,
    'database': 'nursimed_db',
    'user': 'postgres',
    'password': 'YOUR_PASSWORD_HERE'  # رمز عبور خود را اینجا بگذارید
}
```

### 3. اجرای سرور

```bash
cd /workspace/nursimed_v2
python app.py
```

سرور روی پورت 8000 اجرا می‌شود:
```
Server running at http://localhost:8000
```

### 4. باز کردن سایت

مرورگر خود را باز کنید و به آدرس زیر بروید:
```
http://localhost:8000
```

## صفحات سایت

- **صفحه اصلی** (`/`) - نمایش معرفی سایت، محصولات و مقالات
- **مقالات** (`/articles`) - لیست تمام مقالات
- **تک مقاله** (`/article/{id}`) - مشاهده جزئیات مقاله
- **فروشگاه** (`/shop`) - لیست محصولات با فیلتر دسته‌بندی
- **تک محصول** (`/product/{id}`) - مشاهده جزئیات محصول و افزودن به سبد
- **سبد خرید** (`/cart`) - مدیریت سبد خرید و پرداخت
- **درباره ما** (`/about`) - اطلاعات درباره نرسیمد
- **تماس با ما** (`/contact`) - فرم تماس
- **اعضای تیم** (`/team`) - معرفی تیم
- **ورود** (`/login`) - ورود کاربران
- **ثبت نام** (`/register`) - ثبت نام کاربران
- **پادکست‌ها** (`/podcasts`) - به زودی
- **دوره‌ها** (`/courses`) - به زودی
- **خطاها** (`/error400`, `/error500`) - صفحات خطا

## نحوه کار با سبد خرید

1. در صفحه فروشگاه یا صفحه محصول، روی "افزودن به سبد خرید" کلیک کنید
2. آیکون سبد خرید در هدر، تعداد محصولات را نشان می‌دهد
3. با کلیک روی آیکون سبد خرید، به صفحه سبد خرید می‌روید
4. می‌توانید تعداد محصولات را تغییر دهید یا حذف کنید
5. دکمه "پرداخت" را بزنید تا سفارش ثبت شود

## نکات مهم

- جداول دیتابیس به صورت خودکار هنگام اولین اجرای سرور ساخته می‌شوند
- داده‌های نمونه (محصولات و مقالات) به صورت خودکار اضافه می‌شوند
- session ID برای سبد خرید در localStorage مرورگر ذخیره می‌شود
- رمز عبور کاربران به صورت plain text ذخیره می‌شود (برای پروژه دانشجویی مناسب است)

## تغییر تنظیمات دیتابیس

برای تغییر مشخصات دیتابیس، فقط فایل `db_config.py` را ویرایش کنید:

```python
DB_CONFIG = {
    'host': 'localhost',      # آدرس سرور دیتابیس
    'port': 5432,             # پورت دیتابیس
    'database': 'nursimed_db', # نام دیتابیس
    'user': 'postgres',       # نام کاربری
    'password': 'your_password' # رمز عبور
}
```
